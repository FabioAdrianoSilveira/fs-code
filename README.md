# Food Saver

![project status](https://img.shields.io/badge/Development-PROGRESSING-1F6F5F)
![license](https://img.shields.io/badge/License-MIT-6FCF97)

Food Saver é um aplicativo web de E-Commerce focado em venda de alimentos perecíveis próximos à validade. O objetivo do sistema é ser uma plataforma que conecta lojistas buscando vender mais barato para evitar a perda de alimento perecível a clientes que buscam comprar esses alimentos com preços mais acessíveis, assim ajudando ambas as partes enquanto combate o desperdício.

## :hammer_and_pick: Ferramentas e tecnologias

Este projeto possui [React Native](https://reactnative.dev/) como tecnologia para frontend e [Java Spring Boot](https://www.ibm.com/br-pt/think/topics/java-spring-boot) para backend, tendo o banco de dados desenvolvido com [MariaDB](https://mariadb.org/about/). Durante todo o projeto, é utilizado o processo [SpinOff](http://arum.tec.br/SpinOff/index.htm)  para controlar as fases e tarefas, adotando a abordagem de desenvolvimento [Test Driven  Development (TDD)](https://www.ibm.com/br-pt/think/topics/test-driven-development).
